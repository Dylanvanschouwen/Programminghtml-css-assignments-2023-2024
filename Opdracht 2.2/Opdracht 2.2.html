je hebt ook de IMG tag, dat staat voor image tag. je kan met deze tag een plaatje laten zien op je website.

Dit maakt dan weer gebruik van een Table tag, Met tables kan je dingen makkelijker in een tabel zetten.

Dit wordt dan weer gedaan door tr tags en td tags. tr staat voor table row, hierin die je dan td tags zodat je de rij kunt vullen.

td staat voor table cells, dit is eigenlijk het content wat je in een tr tag zet.

ul tags is unordered list. hiermee kan je makkelijk een lijst maken op je website.

Er zijn ook ol tags, ol staat voor ordered list. Je kan hiermee lijsten maken die met nummers geordend zijn.

  3. ol veerbeeld 3
