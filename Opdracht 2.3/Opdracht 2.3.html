Waarom noem je de eerste HTML-pagina; index.html?

De naam index.html wordt vaak gebruikt als de standaardnaam voor de eerste HTML-pagina van een website omdat veel webservers automatisch zoeken naar een bestand genaamd index.html wanneer een iemand de hoofdpagina van een website opent.

Met welke HTML versie werken we nu?

We werken momenteel met HTML 5.3, de nieuwste versie van HTML.

Met welke CSS versie werken we nu?

W3.CSS is de meest recente belangrijke versie van CSS en biedt veel geavanceerde opmaakmogelijkheden voor webpagina's.

Hoe kan het dan een website er net iets anders uitzien in een andere browser? (Edge, Chrome, Firefox, Opera)

Websites kunnen er anders uitzien in verschillende browsers vanwege de manier waarop elke browser CSS en HTML interpreteert en weergeeft.

Welke browser is het populairst en hoeveel procent RAM gebruikt deze browser?

Op dit moment is Google Chrome vaak een van de populairste browsers. Wat betreft het RAM-gebruik, bij elke versie van Google chrome verschilt dit weer.
